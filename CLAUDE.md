# Project: scaleway-starter-kit

## Overview
- Demo/learning project for exploring the Scaleway cloud platform
- Uses Terraform for infrastructure as code
- Includes a Kubernetes cluster among the infrastructure resources
    
## Compliance Documentation
- `COMPLIANCE.md` (English) and `COMPLIANCE.fr.md` (French) document how this project meets sovereign cloud and regulatory requirements
- When making infrastructure changes, update both compliance files if the change affects security, networking, encryption, data residency, or audit capabilities
